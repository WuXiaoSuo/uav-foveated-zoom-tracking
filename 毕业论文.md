# 毕业论文方案：面向无人机微小目标追踪的仿生变焦主动感知方法研究

## 1. 研究背景与约束

本课题面向北京理工大学控制工程专业硕士毕业设计，研究对象为低空开放操场场景下的无人机主动感知与微小目标追踪。

硬件平台：

- 无人机：DJI M400。
- 载荷：DJI Zenmuse H30。
- 机载边缘平台：DJI Manifold 3。

核心资源约束：

- M400 + H30 + Manifold 3 的可用窗口约 3 个月。
- 外场实机采集主要集中在周末。
- 周中主要使用 M 系列 MacBook Air 进行轻量开发，必要时租用 AutoDL 云服务器训练。
- 最终算法需要在 Manifold 3 上部署，关注推理延迟、功耗和稳定性。

研究边界：

- 无人机本体不做自主控制，由飞手手飞或悬停。
- 论文核心控制对象为 H30 的光学变焦倍率。
- 云台控制作为后续扩展，当前不作为最低可行目标。

## 2. 核心科学问题

低空 UAV 微小目标追踪存在视场与分辨率矛盾：

- 广角视场覆盖大，但目标像素面积小，检测和跟踪不稳定。
- 长焦视场目标清晰，但视场窄，目标容易出画或丢失。
- 边缘算力限制下，不能长期对高分辨率全图做重型推理。

本课题将该问题表述为：

> 在无人机本体不参与自主控制、边缘算力受限和外场数据有限的条件下，如何通过主动调节光学变焦倍率，提高低空 UAV 微小目标的可观测性和持续跟踪稳定性？

## 4. 技术路线

整体系统流程：

```text
H30 视频流
  -> Manifold 3 解码/取帧
  -> 轻量目标检测
  -> 目标跟踪与状态估计
  -> 不确定性评估
  -> 主动变焦策略
  -> H30 zoom 控制
  -> 日志与视频存档
```

系统状态机：

```text
Search -> Lock -> Zoom -> Track -> Recover
```

状态含义：

- Search：低倍率搜索，保持较大视场。
- Lock：连续若干帧检测到目标，确认目标。
- Zoom：根据目标尺度和不确定性调节倍率。
- Track：维持目标处于合适像素尺度。
- Recover：目标丢失后主动降低倍率扩大视场，进行重捕获。

### 第 4 章 仿生变焦主动感知方法

核心内容：

- 目标状态定义。
- 检测置信度、像素面积、跟踪丢失帧数等不确定性建模。
- 目标尺度保持的变焦控制律。
- 变焦过程中的目标重捕获和重关联。
- 状态机设计与伪代码。

建议公式：

```text
U_t = λ1(1 - c_t) + λ2 max(0, A_min - A_t) + λ3 e_t + λ4 min(l_t / L, 1)
```

其中：

- c_t：检测置信度。
- A_t：目标归一化像素面积。
- e_t：目标中心偏离画面中心的程度。
- l_t：连续丢失帧数。

尺度保持控制：

```text
z*_t = clamp(z_t * sqrt(A_ref / A_t), z_min, z_max)
z_{t+1} = z_t + clip(z*_t - z_t, -Δz, Δz)
```

### 第 5 章 实验与边缘部署

实验分四类：

1. 公开数据集 baseline：VisDrone、UAVDT、UAV123。

3. Manifold 3 边缘部署。
4. 外场悬停/低速手飞实验。

对比方法：

- 固定 1x。
- 固定 5x/10x。
- 人工变焦。
- 本文主动变焦策略。

评价指标：

- Tracking success rate。
- Lost frame ratio。
- Target average pixel area。
- Recovery time。
- End-to-end latency。
- FPS on Manifold 3。
- Zoom response time。

### 第 6 章 总结与展望

总结：

- zoom-only 载荷级主动感知系统。
- 真实 UAV 平台数据采集与边缘部署。
- 小规模但可复现的数据集和实验协议。

展望：

- 云台-变焦联合控制。
- UAV 本体运动规划。
- 多目标主动感知。
- 更复杂场景和多模态传感。

## 6. 三个月研发计划

目标截止时间：2026-09-15 前形成 ICRA workshop 投稿材料、代码原型和实验数据。

### 第 1-2 周：PSDK 台架验证

- 跑通 H30 视频取流。
- 跑通 zoom 控制。
- 记录 zoom factor、时间戳、视频帧和系统日志。
- 保存原始视频，验证回放。

### 第 3 周：离线 baseline

- 在 VisDrone / UAVDT / UAV123 上跑 YOLO + tracker。
- 对比 YOLOv8n、YOLOv8s、YOLO11n 等轻量模型。
- 对比 ByteTrack、BoT-SORT、OC-SORT。

### 第 4 周：第一次外场采集

- 只做固定倍率采集。
- 目标：人、标志桶。
- 高度：20 m、40 m。
- 倍率：1x、5x、10x。

### 第 5 周：数据清洗与标注

- 统一视频和日志格式。
- 抽帧标注。
- 建立回放器。

### 第 6 周：主动变焦策略 v1

- 实现状态机。
- 实现尺度保持控制律。
- 在离线回放中调参。

### 第 7 周：第二次外场采集

- 测试主动变焦闭环。
- 记录延迟、丢帧、zoom 响应时间。

### 第 8 周：算法增强

- 加入 Recover 状态。

### 第 9 周：第三次外场采集

- 做完整对比实验。

### 第 10 周：Manifold 3 部署优化

### 第 11 周：补实验与消融
